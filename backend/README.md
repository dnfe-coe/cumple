# Backend Cumpleañeros TIGRES-COE

Sistema automático de notificaciones push para cumpleaños institucionales.

## Instalación local

npm install  
npm start

## Variables de entorno

VAPID_PUBLIC=xxxxx  
VAPID_PRIVATE=yyyyy

## Funcionalidad

- Cron diario 7:00 AM
- Revisión automática de cumpleaños
- Envío de notificaciones push
- Sin corrección horaria incorrecta

Sistema desarrollado para Base TIGRES-COE Región Norte.
